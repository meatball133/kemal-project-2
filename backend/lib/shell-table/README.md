# shell-table

Create tables in your shell.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  shell-table:
    github: jwaldrip/shell-table.cr
```

## Usage

```crystal
require "shell-table"
```

See the `/examples` folder for usage (for now)
